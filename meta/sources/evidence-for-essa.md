# Evidence for ESSA

## Discovery

- **Method:** WordPress REST API
- **Endpoint:** `https://evidenceforessa.org/wp-json/wp/v2/program?per_page=100&page={N}&_fields=id,slug,title,link,acf`
- **Total:** 1,297 programs (202 with evidence ratings)
- **Platform:** WordPress with ACF (Advanced Custom Fields)
- **Pages:** 13 (at 100 per page)

## Access

- **robots.txt:** Only `/wp-admin/` blocked. No crawl-delay.
- **llms.txt:** Exists (generated by AIOSEO). Confirms 500+ programs.
- **Authentication:** None required.
- **Sitemap:** `program-sitemap.xml` and `program-sitemap2.xml` exist but the API is far superior.

## Scope

Education programs rated by ESSA evidence strength. Four tiers:

| Tier | Count |
|------|-------|
| Strong | 84 |
| Moderate | 36 |
| Promising | 82 |
| No Evidence | 1,093 |

Current indexing: 78 entries (Strong + Moderate). Remaining: 42 from Strong/Moderate, 82 from Promising = 124 new entries with evidence ratings.

Six subject areas: Reading, Math, Social-Emotional, Attendance, Science, Family Engagement.

## Entry metadata (from API ACF fields)

| Field | ACF key | Notes |
|-------|---------|-------|
| Program name | `title.rendered` | |
| URL | `link` | Full URL |
| Evidence tier | `acf.evidence_rating` | Strong, Moderate, Promising, No Evidence, N/A |

| Number of studies | `acf.number_of_studies` | |
| Number of students | `acf.number_of_students` | |
| Outcomes | `acf.program_outcomes` | HTML |
| Provider | `acf.provider_name` | |
| Designed for | `acf.programs_designed_for` | Array |

## Scraping instructions

**Single pass via API.** 13 requests at 100/page gets everything. Filter client-side by `acf.evidence_rating` for desired tiers. All metadata needed for indexing is in the API response — no individual page fetches required.

## Quirks

- `program_description` and `program_outcomes` are HTML — strip tags for blurb
- The site homepage shows "down for improvements" but API and all pages work fine
- HTML listing pages have unreliable URL-based filtering (GET params don't work) — use API instead
- `_fields` parameter works to reduce payload size
