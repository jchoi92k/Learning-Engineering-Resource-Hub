| 410 | [Duolingo Shared Task on Second Language Acquisition Modeling (SLAM)](https://sharedtask.duolingo.com/2018.html) | dataset | The 2018 Duolingo Shared Task on Second Language Acquisition Modeling (SLAM) provided participating teams with transcripts from millions of exercises completed by thousands of Duolingo learners over their first 30 days of study, annotated at the token level for mistakes, with the task of predicting future learner errors. Each record represents a single lesson or practice session and includes recall accuracy, timing, student and language identifiers, and word-level history of exposures and correct answers (12 columns total). The dataset is available as a 361 MB gzipped CSV and is hosted on Harvard Dataverse. Code is available at https://github.com/duolingo/halflife-regression.
